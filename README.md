# Shared [`tsconfig.json`](tsconfig.json)

## Installation

To install the module use the following commands:

```bash
$ yarn add @pascaliske/tsconfig
```

## Usage

`tsconfig.json`

```json
{
  "extends": "@pascaliske/tsconfig"
}
```
